# Coffee Website ☕

This project is a responsive Coffee website developed using React.js.

## 🚀 Features
- Responsive design
- Modern UI
- Reusable React components
- Coffee menu section
- Services section
- Testimonials
- Reservation form
- Footer with contact information

## 🛠️ Technologies Used
- React.js
- HTML5
- CSS3
- JavaScript

## ▶️ Run Project

Install dependencies:

npm install

Run the project:

npm start

Open in browser:

http://localhost:3000
